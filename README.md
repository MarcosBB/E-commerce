# E-commerce
 
## 📝 Sobre
O Cat House é um site de vendas de produtos para gatinhos feito em `python` com o  framework `django` e banco de dados `postgreSQL` .
Os arquivos estáticos e as fotos dos produtos são hospedados no `AWS S3`.
